    Consider MoneyStream

    ❯ The internet is a streaming medium. MoneyStream is streamable money for the streaming web
    ❯ Each MoneyStream monetized browser session settles to one transaction on the bitcoin ledger
    ❯ 5 billion current web users on the planet engaged in 200 monetized sessions per day equates to 1 trillion potential transactions per day using MoneyStream
    ❯ Use MoneyStream when you don't know how long you will be browsing content. Examples of such sessions can be: watching videos, gaming, or interacting with any content including simply reading a web page. MoneyStream monetizes virtually all most common browser sessions.
    ❯ MoneyStream pays websites to remove ads from a browsing session
    ❯ MoneyStream pays content creators and platforms to access streamable content using a pay-as-you-go model. There are no burdensome subscriptions to maintain. The user always controls the funds in their wallet.
    ❯ The user is happy paying for a clean content stream free from tracking ads and extraneous noise.
    ❯ MoneyStream allows content platforms to be easily created. One line of HTML code can enable monetization on a web site.
    ❯ MoneyStream provides an opportunity for everyone to earn bitcoin from streaming web content.

    How does MoneyStream apply to the BSV hackathon theme for a global "record of events"?

    ❯ MoneyStream monetizes events inside the browsers of a global internet population. MoneyStream measures the exchange of value, captures the browser events and records them on bitcoin at the end of the each browser session. MoneyStream is a monetizer of a global record of events.
    ❯ The bitcoin miners protect the web sites from double spends, providing merchants a safe operating environment to distribute content to anyone in the world. The web content provider has assurance that they will get paid at the end of the user browser session.
    ❯ The bitcoin protocol provides for safe and honest realtime exchange of value
    ❯ A transaction id is a globally unique pointer to the record of events and acts as a receipt for cash payments that can be accepted by any vendor. Vendors can verify cash payment from anyone anywhere in the world. Instantly.
